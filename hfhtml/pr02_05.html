HTML таблица

Tag Atribute Description
html lang Обязательный, для лучшей доступности, указывает язык используемый на странице, например: lang="ua".
xmlns Определяет namespace-XML для документа. Обычно используется в XHTML. Значение по умолчанию "http://www.w3.org/1999/xhtml"
dir Указывает направление текста на странице. Значения: ltr, rtl
manifest Устаревший атрибут, раньше использовался для указания манифеста содержащего информацию для работы веб-приложения офлайн(часть технологии AppCache). В современных браузерах заменён service workers.
head - Используется для размещения мета-данных веб-страницы, таких как заголовок страницы, ссылки на стили, скрипты и другая информация, которая не отображается непосредственно на странице. Он распологается внутри корневого тега и передаёт информацию браузеру о том, как обрабатывать страницу(содержимое).
